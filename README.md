# probot-workflow

> A GitHub App built with [Probot](https://github.com/probot/probot) that A probot app to automate workflow.

## Setup

```sh
# Install dependencies
npm install

# Run with hot reload
npm run build:watch

# Compile and run
npm run build
npm run start
```
